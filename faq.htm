VASL 4.0 FAQ

Q:  Where do I set the server address?

A:  Nowhere. VASL 4.0 determines the server address dynamically at run-time.

Q:  I'm unable to connect to the server. My ISP connection goes through a proxy host. How can I connect?

A:Specify the proxy host name and port in the Web Start control panel. If that doesn't work, use the Manual installation and start VASSAL from a DOS shell with the command: 

java -jar runVASSAL.jar -Dhttp.proxyHost=<your proxy host name>
            -Dhttp.proxyPort=<your proxy port>
