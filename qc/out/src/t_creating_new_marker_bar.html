Creating a new QC bar

You can create new QC bars for your interface. For example, you might want to create a custom QC bar to use when playing a scenario with Night rules in effect, or for an airborne or amphibous landing.

Note: You cannot customize the built-in QC bar configuration. You must create a new QC bar and customize that.

To create a new QC bar:

  1. Click .
    VASL displays the Select popup menu.
  2. Select the QC configuration you want to copy.
  3. Click Copy current QC configuration..
    VASL closes the popup and creates a copy of the QC bar with the name "copied_QC_bar"(copy). For example, if you copy the built-in configuration, the new configuration is named "Built-in configuration (copy)".
