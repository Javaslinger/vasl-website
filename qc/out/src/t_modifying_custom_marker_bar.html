Modifying a QC bar

You can modify custom QC bars.

Note: You cannot customize the built-in QC bar configuration. You must create a new QC bar and customize that.

You can add both counter and overlays to a QC bar.

To modify a QC bar:

  1. Click .
    VASL displays the Select popup menu.
  2. Select the QC configuration you want to modify.
    The configuration is selected and the menu disappears.
  3. Click .
    VASL displays the Select popup menu.
  4. Click Modify current QC configuration..
    VASL displays the Edit: configuration dialog.
  5. To add a new counter or overlay to the QC bar, drag the counter or overlay from the tray to the Edit: configuration dialog.
  6. To move an item up or down, select the item and click the up and down arrows in the button bar.
  7. To add a new menu or submenu item, click the
    If you do not have an item selected, a new menu item is added. If you have an item selected, a submenu item is added to the selected item.
    A new blank menu item is added.
  8. To promote a submenu item to the next higher menu, select the item you want to move and click the icon.
  9. You can change the label on any item, as well as the label of the QC bar as a whole. To change the label on a menu item or the QC bar:
    1. Select the item whose label you want to change.
    2. Click icon.
      VASL displays the Input dialog with the current test of the label.
    3. Modify the text in the Input dialog.
    4. Click OK.
    The label of the item is updated with the new text.
  10. To delete an item, select the item you want to delete and click the icon.
  11. To undo an action, click the icon.
All changes are applied immediately.
