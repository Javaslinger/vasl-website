Deleting a QC bar

You can delete custom QC bars.

Note: You cannot delete the built-in QC bar configuration.

To delete a custom QC bar:

  1. Click .
    VASL displays the Select popup menu.
  2. Select the QC configuration you want to delete.
    The configuration is selected and the menu disappears.
  3. Click .
    VASL displays the Select popup menu.
  4. Click Delete current QC configuration..
    VASL displays a confirmation dialog.
  5. Click Yes.
